BAI Browser Audit Inventory

v0.1.0

Captures artifacts from the currently signed-in Chrome profile, hashes each one with SHA-256, and writes a verifiable evidence package to a folder you choose, including a removable drive.

This is a live, single-profile acquisition. It reads only this profile through Chrome's own APIs. It cannot reach saved passwords, deleted records, other accounts on this device, or data outside the browser, and the act of running it is itself part of the profile state. Timestamps come from this device's clock, which is not a trusted timestamp authority.
This browser does not expose the File System Access API, so the package cannot be written to a chosen folder here. Open this console in a recent desktop Chrome / ChromeOS build.
